Eventy Pro PWA

Scanne le QR code pour installer l'app sur ton téléphone

Entrez l'URL ci-dessus puis cliquez Générer

📱 Option 1 : Test local (même Wi-Fi)

  1. Ouvre un terminal dans le dossier pro-pwa/
  2. Lance :
    node server.js
  3. Note l'adresse réseau affichée (ex: http://192.168.1.x:3000)
  4. Entre cette adresse ci-dessus et clique Générer QR
  5. Scanne le QR code avec ton téléphone (même réseau Wi-Fi)
  6. Sur iOS : Safari → bouton Partager → "Sur l'écran d'accueil"
  7. Sur Android : Chrome → menu ⋮ → "Installer l'application"

🚀 Option 2 : Déployer sur Vercel (accès partout)

  1. Installe Vercel CLI :
    npm i -g vercel
  2. Dans le dossier pro-pwa/ :
    3. cd pro-pwa && vercel deploy --prod
  3. Copie l'URL Vercel donnée et entre-la ci-dessus
  4. Scanne le QR code pour installer la PWA !

⚡ Option 3 : Déployer sur Vercel via GitHub

  1. Pousse le dossier pro-pwa/ sur un repo GitHub
  2. Va sur vercel.com/new → Importer le repo
  3. Configure : Output Directory = . (racine)
  4. Déploie et utilise l'URL pour le QR code
