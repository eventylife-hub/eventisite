2. SENSITIVE DATA IN STORAGE
   Status: ✅ SECURE
   
   Token Storage (EXCELLENT):
   - Authentication tokens stored in httpOnly cookies (NOT localStorage)
   - Middleware retrieves from secure cookies only
   - getToken() methods return null — tokens managed server-side
   - Protects against XSS attacks accessing tokens via JavaScript
   
   Non-Sensitive Data in localStorage (ACCEPTABLE):
   - Only safe UI preferences stored:
     * eventy_user (public profile data)
     * eventy_ui_state (UI state)
     * eventy_preferences (user settings)
   - SSR-safe wrapper functions prevent window access during SSR
   
   Files Reviewed:
   - /lib/api-client.ts (lines 21-92)
   - /middleware.ts (line 140)
   - /lib/constants.ts (lines 172-176)
   - /lib/utils.ts (lines 291-313)
