## 1. Système d'Alliances Eventy

### Concept
Une Alliance = groupement de 5-20 caravanes qui s'unissent sous une bannière commune.
Objectif : objectifs gaming à très grande échelle, événements IRL, communautés thématiques.

### Structure
```
Alliance (500-3000 membres)
 ├── Caravane A (100-200 membres)
 │    ├── Tribu A1 (5-15)
 │    └── Tribu A2 (5-15)
 ├── Caravane B
 └── Caravane C...
```

### Grand Maître d'Alliance
- Élu par les Meneurs des caravanes membres (vote)
- Pouvoirs : représentation externe, organisation des Guerres d'Alliances
- Pas de pouvoirs internes sur les caravanes individuelles

### Pages (futur)
- `/jeux/alliances` — hub des alliances
- `/jeux/alliances/[id]` — détail alliance
- `/jeux/alliances/[id]/guerre` — Guerre d'alliances

## 2. Partenariats Jeux Vidéo

### Vision
Eventy s'interface avec des jeux vidéo existants pour cross-promouvoir le voyage.

### Pistes de partenariats

#### Jeux mobiles voyage/exploration
- **Pokémon GO** : Les caravanes Eventy ont des événements géolocalisés liés aux destinations
- **Geocaching** : Intégration native, les voyages Eventy incluent des caches exclusives
- **Ingress** / jeux AR : Portails exclusifs dans les destinations Eventy

#### Jeux PC/Console
- **No Man's Sky** : Caravanes Eventy = guildes dans le jeu, événements croisés
- **Sea of Thieves** : Aventures maritimes → promotions voyages maritimes Eventy
- **Euro Truck Simulator** : Co-branding "Conduis virtuellement, voyage pour de vrai"

#### Jeux compétitifs
- Tournois e-sport parrainés par Eventy (récompenses = voyages)
- Top classeurs de jeux populaires → prix voyage Eventy

### Mécanisme cross-promotion
```
Gagner dans le jeu partenaire → débloquer un héros/badge/réduction Eventy
Voyager avec Eventy → débloquer skin/item/titre dans le jeu partenaire
```

## 3. Championnats Gaming Sponsorisés

### Format
- Saisonniers (tous les 3 mois)
- Multi-caravanes (ou multi-alliances)
- Sponsor titre (ex : "Championnat SNCF" ou "Decathlon Cup")
- Livestreamés sur Twitch/YouTube

### Récompenses
- 1ère caravane : voyage offert pour toute la caravane (ex : Andalousie 7j)
- 2ème-3ème : réductions importantes
- Meilleur Meneur : titre spécial + avantages Meneur Pro
- Prix individuels : héros légendaires, XP massif

### Business model
- Sponsor paie pour le naming et la récompense
- Eventy garde la visibilité + la base de contacts gaming
- Win-win : sponsor touche une audience jeune et mobilisée

## 4. Robot / Mascotte de Caravane (futur)

### Concept
Chaque caravane pourra avoir un **robot/mascotte** personnalisé :
- IA simple qui anime le chat de la caravane
- Rappelle les raids, les défis, les événements
- Peut commenter les résultats ("GG les gars, raid réussi !")
- Personnalité choisie par le Meneur (sérieux/fun/épique)

### Noms de mascotte possibles
- "Nomad" (bot officiel Eventy)
- Personnalisable par la caravane

### Technologie
- Webhooks → bot Discord-like dans le chat caravane
- Messages automatisés basés sur événements
- Futur : LLM léger pour réponses contextuelles
