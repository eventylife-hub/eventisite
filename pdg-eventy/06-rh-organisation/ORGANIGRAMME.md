## Fiches de poste clés

### Chargé(e) de voyages
- **Mission** : Organiser les voyages de A à Z, relation client, logistique
- **Compétences** : BTS Tourisme ou expérience équivalente, sens du service, organisation, maîtrise outils numériques
- **Contrat** : Freelance (Phase 2) puis CDI (Phase 3)
- **Rémunération** : 1 800€ - 2 500€ net/mois (freelance) ou 2 200€ brut/mois (CDI)
- **KPIs** : Taux satisfaction > 4,5/5, délai réponse < 24h, 0 erreur logistique

### Commercial(e) B2B
- **Mission** : Prospecter et signer des clients B2B (CE, PME, séminaires)
- **Compétences** : Expérience commerciale B2B 2+ ans, réseau tourisme/événementiel apprécié
- **Contrat** : Freelance (fixe + variable)
- **Rémunération** : 1 000-1 500€ fixe + **8-10% commission sur marge** des contrats signés
- **KPIs** : 3 séminaires signés/mois, CA B2B > 30% du total, marge > 25%

### Développeur fullstack
- **Mission** : Maintenance et évolution de la plateforme Eventy (290 000+ lignes)
- **Compétences** : NestJS, React/Next.js, PostgreSQL, Prisma, TypeScript, tests automatisés
- **Contrat** : CDI ou freelance senior
- **Rémunération** : 3 000€ - 4 500€ net/mois (3 800€ brut CDI Paris)
- **KPIs** : Uptime > 99,9%, couverture tests > 80%, temps réponse API < 200ms

### Réseau auto-entrepreneurs (apporteurs d'affaires)

- **Mission** : Apporter des clients (groupes, associations, entreprises) à Eventy
- **Statut** : Auto-entrepreneur, contrat d'apporteur d'affaires
- **Rémunération** : 5-8% de la marge brute sur chaque affaire apportée
- **Exemple** : Voyage de groupe 6 045€, marge 1 371€ → commission AE : 69-110€
- **Avantage** : Aucun coût fixe pour Eventy, réseau scalable
- **Objectif Phase 2** : 5-10 auto-entrepreneurs actifs
- **Objectif Phase 3** : 10-20 auto-entrepreneurs actifs

## Process internes

### Nouveau voyage — Workflow (10 étapes)

1. **Demande client** → formulaire en ligne eventylife.fr ou email contact@eventylife.fr
2. **Qualification** → nombre de personnes, dates, budget, destination, type (B2C/B2B)
3. **Sourcing** → sélection prestataires dans le catalogue (module HRA backend)
4. **Devis instantané** → génération via la plateforme (module documents) — **inclut Pack Sérénité**
5. **Validation client** → signature électronique + acompte 30% via Stripe
6. **Confirmation prestataires** → réservations fermes (hébergement, transport, activités)
7. **Suivi** → relance solde J-30, envoi documents voyage J-15
8. **Jour J** → contact d'urgence disponible 24h/24 (assistance Pack Sérénité)
9. **Post-voyage** → enquête satisfaction (email automatique J+3)
10. **Fidélisation** → relance avis Google/Trustpilot (J+7), offre parrainage (-50€)

### Nouveau partenaire — Workflow (8 étapes)

1. **Identification** → recherche web, salons tourisme, bouche-à-oreille, annuaires
2. **Premier contact** → email template (voir EMAILS-PARTENAIRES.md) — **mentionner volume 500-5 000 voyageurs/an**
3. **Qualification** → grille d'évaluation (note min 3,5/5)
4. **Devis test** → cas concret (voyage type de 15-40 personnes)
5. **Négociation** → contrat cadre annuel avec remise volume (-10% à -15%)
6. **Intégration** → ajout dans la plateforme (module HRA), fiche partenaire complète
7. **Test réel** → premier voyage avec ce partenaire
8. **Évaluation** → retour client + note interne → décision de poursuite ou remplacement
