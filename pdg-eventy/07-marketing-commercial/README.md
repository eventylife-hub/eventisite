## 📚 Structure — Les 4 fichiers clés

### 1. BRAND-GUIDE-RAPIDE.md (394 lignes, 11 sections)
**La bible du brand Eventy.** À consulter avant chaque livrble marketing.

**Sections** :
1. Les couleurs — Palette exacte + utilisation (orange CTA, bleu texte, vert succès)
2. Typographie — Fonts exactes + hiérarchie (Fraunces titres, Outfit body)
3. Messagerie clé — 4 piliers Eventy (bus complet, pack sérénité, humain, unique)
4. Ton de voix — DO/DON'T tone (ami vs formel, transparent vs vague)
5. Règles visuelles — Photos (visages happy, pas paysages vides)
6. Templates emails — Structure type (hero, piliers, tarifs, FAQ, CTA)
7. Copywriting règles — Lexique Eventy vs jargon touristique
8. Stratégie réseaux — Tone par plateforme (Instagram casual, LinkedIn pro)
9. Landing pages — Structure universelle (hero → 4 piliers → tarifs → FAQ → CTA)
10. Checklist qualité — Points avant publication
11. Recap — 3 règles d'or

**Quand consulter** : Avant tout email, post social, page landing, visuel marketing

## 🎨 Cheat Sheet — Couleurs & Fonts

### Copier-coller facilement

**CSS Colors** :
```css
--color-orange: #FF6B35;    /* CTAs, energy */
--color-blue: #0077B6;      /* Text, trust */
--color-green: #06D6A0;     /* Success */
--color-gray: #0A1628;      /* Body text */
```

**CSS Fonts** :
```css
--font-title: 'Fraunces', serif;
--font-body: 'Outfit', sans-serif;
```

**Button CTA** :
```css
background: #FF6B35;
font: 600 Outfit;
color: white;
border-radius: 8px;
padding: 12px 24px;
```

**H1 Title** :
```css
font: 600 Fraunces;
font-size: 36-48px;
line-height: 1.2;
color: #FF6B35;
```
