# Grille Tarifaire — Eventy

> **Dernière mise à jour** : 6 mars 2026
> **Stratégie** : Bus complet 53 places = règle n°1 | Assurance incluse = force de vente n°1
> **Principe** : Chaque voyage est dimensionné pour remplir un bus complet (53 places). Le prix par personne est optimisé grâce au taux de remplissage.

---

## Modèle de revenus

Eventy combine quatre sources de revenus :
1. **Commission sur les voyages organisés** (revenu principal)
2. **Frais de service / dossier** (revenu récurrent par voyageur)
3. **Marge sur assurance voyage** (Pack Sérénité Eventy — ~50% de marge)
4. **Revenus SaaS** (Phase 2 — abonnements Pro)

---

## 1. Commission voyages — Barème standard

| Type de prestation | Commission Eventy | Notes |
|--------------------|-------------------|-------|
| Hébergement | 10-15% | Négociable volume (53 pers = levier fort) |
| Transport autocar | 8-12% | **Coût fixe bus ~1 000€/jour** → marge augmente avec le remplissage |
| Vols (allotement) | 5-8% | Via TO/consolidateur, 53 places = allotement standard |
| Vols (charter A320) | 10-15% | À partir de 159 voyageurs (3 bus), -35 à -50% vs allotement |
| Activités | 15-25% | **Meilleure marge** — prioriser, négocier tarif groupe 50+ |
| Restauration | 10-15% | Menu groupe négocié, tarif dégressif dès 40+ couverts |
| Assurance voyage | Intégrée au prix | Pack Sérénité (marge ~50% sur prime) |

### Impact du remplissage bus sur le coût transport/personne

| Remplissage | Voyageurs | Coût transport/pers (1 jour) | Impact marge |
|-------------|-----------|------------------------------|--------------|
| 50% | 27 pers | 37,04€ | ❌ Marge compressée |
| 75% | 40 pers | 25,00€ | ⚠️ Correct |
| **100%** | **53 pers** | **18,87€** | **✅ Marge optimale** |
| 2 bus pleins | 106 pers | 18,87€ + tarif dégressif | ✅✅ Volume négociable |
| 3 bus pleins | 159 pers | ~17€ (tarif dégressif) | ✅✅✅ Charter possible |

> **Règle PDG** : Ne JAMAIS lancer un voyage à moins de 40 personnes (75% remplissage). Objectif : 53 personnes systématiquement.

### Frais de dossier

| Prestation | Montant |
|-----------|---------|
| Frais de dossier (par personne) | 35€ - 50€/personne |
| Frais de modification | 30€ - 50€ |
| Frais annulation (selon barème CGV) | 30% - 100% |
| Frais urgence (réservation J-15 ou moins) | +25% |

> Sur un bus complet : frais de dossier = 35€ × 53 = **1 855€** de revenu pur (100% marge)

### Modèle économique assurance — Bus complet 53 places

| Donnée | Valeur |
|--------|--------|
| Coût prime assureur | ~2-3% du prix voyage |
| Prix intégré au client | ~4,5% du prix voyage |

| Exemple : Séjour Europe 1 185€/pers | Prime : ~36€ → Vendu : ~53€ → Marge : ~18€/pers |

| **Sur un bus complet (53 pers) — Europe** | Marge assurance : **~954€** |
| **Sur 3 bus (159 pers) — Europe** | Marge assurance : **~2 862€** |
| Taux de souscription | ~100% (car inclus dans le prix) |

## 6. Sources de remplissage — Comment atteindre 53 voyageurs

| Source | Part du remplissage | Mécanisme |
|--------|---------------------|-----------|
| **Groupe organisateur** | 50-70% (27-37 pers) | L'organisateur amène son groupe (amis, famille, asso, CE) |
| **Places ouvertes eventylife.fr** | 20-30% (11-16 pers) | Voyageurs individuels qui rejoignent un voyage publié |
| **Réseau auto-entrepreneurs** | 10-20% (5-11 pers) | Auto-entrepreneurs Eventy qui amènent des voyageurs via leur réseau |

### Stratégie multi-groupes (bus partagé)

Si un organisateur ne remplit que 25-30 places, Eventy peut :
1. **Fusionner 2 groupes** compatibles (même destination, même dates) dans 1 bus
2. **Ouvrir les places restantes** sur eventylife.fr
3. **Activer le réseau auto-entrepreneurs** pour compléter

> Exemple : Groupe A (30 pers Bordeaux → Barcelone) + Groupe B (15 pers Toulouse → Barcelone) + 8 places ouvertes = **53 pers, 1 bus complet optimisé**

## 8. Politique de prix

### Principes fondamentaux
- Prix affichés **TTC** (obligation légale Code du Tourisme)
- **Pas de frais cachés** — transparence totale = confiance
- **Assurance INCLUSE** — le client ne paie pas en plus (Pack Sérénité)
- Prix par personne, base chambre double
- **Prix optimisé grâce au bus complet** — le client bénéficie du tarif groupe 53 personnes
- Suppléments clairement indiqués (single, vue mer, etc.)
- Révision de prix limitée aux cas prévus par la loi (carburant, taxes, devises)

### Acompte et paiement
- Réservation : **30% d'acompte**
- Solde : **J-30** avant départ
- Split payment disponible (chacun paie sa part via le module checkout)
- Paiement CB via Stripe : 1,4% + 0,25€ par transaction
- Paiement SEPA via Stripe : 0,35€ par transaction

## 🧠 Décisions PDG (06/03/2026)

- **Bus complet 53 places** = unité de base de TOUS les calculs tarifaires
- **Minimum de lancement** : 40 personnes (75% remplissage) — jamais en dessous
- **Objectif systématique** : 53 personnes par bus
- **Multi-bus** : 106 (2 bus), 159 (3 bus + charter), 212 (4 bus) pour les grands événements
- **Sources de remplissage** : Groupe organisateur + places ouvertes + auto-entrepreneurs
- **Pack Sérénité** reste inclus dans TOUS les prix sans exception
- **B2B prioritaire** : séminaires = meilleure marge (26,8%) + volume garanti
- **Levier négociation fournisseurs** : "Nous amenons 53+ personnes à chaque voyage" = tarifs préférentiels
