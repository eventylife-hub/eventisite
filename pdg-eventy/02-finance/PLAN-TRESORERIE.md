## Flux financiers — BFR favorable (bus complet 53 pers)

```
Client paie 19 766€ TTC (53 personnes × 373€)
├── Acompte 30% = 5 930€ (à la réservation, J-45)
└── Solde 70% = 13 836€ (à J-30 avant départ)

Stripe verse les fonds : J+2 (CB) / J+5 (SEPA)

Eventy paie les prestataires : J+30 après le séjour
├── Hébergement : 5 597€
├── Transport (bus) : 2 400€
├── Activités : 2 968€
├── Restauration : 3 731€
└── Assurance : 451€

→ Trésorerie positive en permanence (client paie avant, fournisseur après)
→ Marge Eventy : 4 919€ (24,9%)
→ Cash disponible avant paiement fournisseurs : ~45 jours
```

## Règles de gestion trésorerie

1. **Trésorerie minimum** : Toujours garder 3 mois de charges fixes en réserve (~3 000€)
2. **Encaissement client** : Acompte 30% immédiat, solde J-30 avant départ
3. **Paiement fournisseurs** : Négocier J+30 minimum (argument : "53 voyageurs par bus")
4. **BFR favorable** : Les clients paient AVANT, les fournisseurs APRÈS = trésorerie positive
5. **Stripe** : Fonds disponibles J+2 (cartes), J+5 (SEPA) → pas de gap de tréso
6. **APST** : Cotisation annuelle renouvelable → provisionner 175€/mois
7. **Provision charter** : À partir de M9, provisionner pour les vols charter (acompte à verser à la compagnie)
