## 🎯 Vision stratégique

> **"Chez Eventy, vous partez l'esprit tranquille. Garantie complète incluse."**

La plupart des agences de voyages proposent l'assurance en option payante. Eventy se différencie en **incluant une assurance complète dans chaque voyage**. C'est notre argument de vente n°1.

## Modèle économique

### Option A : Assurance intégrée au prix (recommandé ⭐)

| Paramètre | Valeur |
|-----------|--------|
| Coût assurance / voyageur | ~4,5% du prix du voyage |
| Prix moyen voyage | 800€/personne |
| Coût assurance moyen | ~36€/personne |
| Intégré dans le prix du voyage | Oui — transparent pour le client |
| Marge Eventy sur l'assurance | Commission 5-10% possible |

**Avantage** : Le client ne se pose pas la question, c'est inclus → zéro friction, argument marketing puissant

### Option B : Assurance optionnelle avec forte incitation

| Paramètre | Valeur |
|-----------|--------|
| Prix affiché | "à partir de 36€/personne" |
| Taux de souscription visé | >80% (grâce à la mise en avant) |
| Intégration parcours d'achat | Case pré-cochée (conformément RGPD) |

## 🧠 Arguments de négociation (décision PDG — 05/03/2026)

### Le volume est notre arme

| Argument | Chiffre |
|----------|---------|
| Voyageurs/an objectif Y1 | 500 - 1 500 |
| Voyageurs/an objectif Y2 | 3 000 - 5 000 |
| Taux de souscription | >80% (intégré au prix) |
| Contrats/an estimés Y1 | 400 - 1 200 |

### Arguments clés

1. **Volume massif garanti** : chaque voyageur Eventy = un contrat d'assurance. Pas d'opt-in/opt-out, c'est systématique.
2. **Risque faible** : groupes encadrés, voyages majoritairement France, organisation professionnelle
3. **Intégration digitale** : API pour automatiser souscription → coût de gestion minimal pour l'assureur
4. **Exclusivité** : prêts à être partenaire exclusif = 100% de la production passe chez eux
5. **Croissance** : plateforme tech en hyper-croissance → volume x3-x5 en 2 ans

### Ce qu'on demande

- **Tarif dégressif** par tranches : <500, 500-2000, >2000 voyageurs/an
- **Couverture "tout compris"** incluant annulation toutes causes
- **Commission Eventy** 5-10% sur chaque contrat
- **Gestion 100% digitale** (API REST, webhook confirmation, certificat PDF auto)
- **Assistance 24h** avec numéro dédié Eventy
- **Co-branding** : "Protégé par [Assureur] x Eventy"
